AI Odyssey — IIT Patna M.Tech Final Project

AI Odyssey: Adaptive Intelligence through Reinforcement Learning

AI Odyssey is an experimental project that explores how artificial agents can learn and adapt to dynamic environments using Deep Reinforcement Learning (DRL). Built with Unity ML-Agents, it features an AI-driven agent that learns optimal behaviors—such as navigation, puzzle-solving, or motion control—through continuous interaction, feedback, and reward-based learning. The project demonstrates how adaptive AI systems can evolve intelligently, making it a foundation for future game AI, robotics, and simulation-based research.
